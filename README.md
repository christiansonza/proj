# Todo App with React, Redux Toolkit (RTK Query) & Express

A simple Todo application built with **React.js** (with Redux Toolkit and RTK Query) for the frontend, and **Express.js** with **Sequelize/MySQL** for the backend. Features include adding, editing, deleting, marking tasks as done, and filtering tasks (all, complete, incomplete).

---

## Features

- Create, read, update, delete todos
- Mark tasks as complete / not complete
- Filter todos by status (all / complete / incomplete)
- Search todos
- Added modal for add/edit tasks
- React frontend with Redux Toolkit + RTK Query
- Express backend with Sequelize ORM and MySQL
- RESTful API
